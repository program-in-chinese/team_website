陶哲轩的 [不等式证明工具](https://github.com/teorth/estimates) 用的是远在这十年AI潮兴起之前的技术、工具和算法，不需任何所谓大模型和训练过程，即便还远未充分优化也在多数情况下具备秒级的响应时间，更无需任何昂贵硬件，不更接近简约低能耗的「智能」吗？

```
开始证明。当前情况：
x: pos_real
y: pos_real
z: pos_real
h1: x < 2*y
h2: y < 3*z + 1
求证：x < 7*z + 2
检验以下不等式可否成立：
1*x > 0
1*x + -2*y < 0
1*y > 0
1*y + -3*z < 1
1*z > 0
1*x + -7*z >= 2
以下式子取和，得证不可能成立：
1*x + -2*y < 0 两边乘 -1/4
1*y + -3*z < 1 两边乘 -1/2
1*z > 0 两边乘 1/4
1*x + -7*z >= 2 两边乘 1/4
证明完毕！
```
